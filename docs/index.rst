SimscapeMSS Docs
================

This documentation is intended to document the Simulink Simscape library developed as a joint effort by `MotionTech AS <https://motiontech.no/>`_ and `cDynamics AS <https://cdynamics.no/>`_ . The Simulink library is building on top of `Thor I. Fossen <https://scholar.google.com/citations?user=Sn8fzegAAAAJ&hl=en&oi=ao>`_'s Handbook of Hydrodynamics :cite:`fossenHydro` and Fossen's public available `Marine Systems Simulator (MSS) GitHub library <https://github.com/cybergalactic/MSS>`_
